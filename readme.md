# Study Quiz

This project was a self-assessment tool to test knowledge while taking courses.
AI integration allows you to generate questions and test your skills on any subject. 
Open AI, deepseek etc. 

## Overview

A simple quiz application to reinforce learning and track progress during course studies.

## Purpose

- Test understanding of course material
- Provide immediate feedback
- Integrate with AI assistance
- Utilize API-driven responses
- Identify areas needing review

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/study-quiz.git
```

2. Install dependencies

```bash
cd study-quiz
npm install
```

3. Configure environment variables

```bash
cp .env.example .env
```

4. Start the application

```bash
npm run start
```
